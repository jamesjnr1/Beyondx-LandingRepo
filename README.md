# BeyondX — Landing Page

Marketing landing page for **BeyondX**, a skills-verified hiring platform connecting
people rebuilding their lives after incarceration with employers across Greater Accra.

Built with Vite + React + TypeScript + Tailwind CSS.

## Brand
- **BeyondX Green** `#6BAB21`
- **BeyondX Dark** `#12180E`
- Logo lockup + reversed (light) variant live in `/public`

## Develop
```bash
npm install
npm run dev
```

## Build
```bash
npm run build      # outputs to /dist
npm run preview    # preview the production build
```

## Deploy (Vercel)
Framework preset: **Vite**. Build command `npm run build`, output directory `dist`.
Pushes to the production branch auto-deploy.
